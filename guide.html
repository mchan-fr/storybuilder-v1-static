Quick Start Guide

1

Understanding the Interface

StoryBuilder has three main panels:

Left

Blocks Panel - Add and reorder story blocks. Drag blocks to rearrange them.

Middle

Properties Panel - Edit the selected block's content, images, and styling.

Right

Preview - See your story as readers will see it. Updates in real-time.

2

Explore the Demo Project

We've included a demo project called "Hiking Mt. Whitney" to help you get started. Load it from the projects dropdown in the top-left of the tool to see a complete story with various block types.

View the finished Mt. Whitney story to see what's possible.

3

Adding Story Blocks

Select a block type from the dropdown and click "+ Add Block". Available blocks:

Full-Bleed Media

Edge-to-edge photos/videos

Gallery

2-4 image layouts

Hero

Big headline + media

Split Layout

Image + text side by side

Text

Paragraphs with formatting

Zoom on Photo

Scroll-triggered zoom

4

Configuring Your Blocks

Click any block in the left panel to select it. The middle panel shows all editable properties:

  • Media paths - Enter paths like projects/my_project/media/photo.jpg
  • Text content - Headlines, body text, captions
  • Styling - Colors, fonts, sizes, alignments
5

Exporting Your Story

When you're ready to publish:

  1. Click "Export HTML" to download a standalone HTML file
  2. Upload the HTML file and your media folder to any web host
  3. Share the URL with your audience!

Tip: Use "Save JSON" to save your project and reload it later with "Load JSON".

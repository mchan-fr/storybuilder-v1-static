Quick Start Guide

8 Steps to Create Your First Project

Note: StoryBuilder requires Chrome or Edge browsers.

Set Up Your Project

1

Click + New in the left column to start a new project. Your Preview pane should turn black.

2

Enter your project name in Page Title at the top (you can call it "Test01" for now).

3

Click Select Media Folder to choose the folder containing your photos and videos. This gives StoryBuilder browser access to your media. Here is the recommended folder structure:

StoryBuilder Projects/
     └── Test01/
         └── media/
            ├── photo1.jpg
            └── video1.mp4
4

Click Save. Your project now appears in the "Stories" dropdown.

Build Your Story

5

Click Select Block Type → Choose Hero-horizontal → Click + Add Block

6

Click block #1. In the column next to it, you should now see options for that block. Open Media and add your hero image by typing the full filename (e.g., photo1.jpg). Click Preview.

7

Open Headline and adjust the font, size, color, etc. Click Preview. Save whenever you like the work you've done.

Add other block types. Try Media-vertical. Once you add it, notice that it becomes block #2 -- this is the process of building your story: block by block. Open #2, add a vertical photo or video. Play with the different options, such as block settings, body text, byline, drop cap, etc.

8

When you're done, click Export HTML to download. Save it to your project folder (next to your media folder). Double-click the HTML file to view locally.

Pro Tip: For web publishing, compress video files using Handbrake (free, open-source) to improve load times.
